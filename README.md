# Social to Lead Agent

## How to Run

```bash
python main.py
```

## Features
- Intent Detection
- RAG-based Knowledge Retrieval
- Lead Capture Tool

## Architecture
This project uses Python and JSON-based RAG.
The chatbot detects user intent and retrieves answers from a local knowledge base.
It also captures leads when users show high purchase intent.

## WhatsApp Integration
This project can be integrated with WhatsApp using Twilio webhooks.
